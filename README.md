# ShadowChat

ShadowChat is a CRUD-based chat application designed to manage conversations efficiently. The project focuses on implementing RESTful routing, database operations, and a structured MVC approach with a clean user interface.

---

## Tech Stack

- Node.js  
- Express.js  
- MongoDB  
- Mongoose  
- EJS  
- CSS  

---

## Features

- Create new chats  
- View all chats  
- Edit messages  
- Delete chats  
- Dark themed UI  

---

## Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/shadowchat.git
```

### 2. Navigate into the project folder
```bash
cd shadowchat
```

### 3. Install dependencies
```bash
npm install
```

### 4. Make sure MongoDB is running locally

### 5. Run the server
```bash
nodemon index.js
```

### 6. Open in browser
```bash
http://localhost:8080/chats
```
